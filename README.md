Giftora – A modern MERN Stack e-commerce platform for gifting and handmade products.

Built with React, Node.js, Express.js, MongoDB, and Tailwind CSS, Giftora offers a smooth shopping experience with responsive design, secure authentication, dynamic product management, and an elegant user interface.

This project enhanced my full-stack development skills and gave me hands-on experience in building scalable web applications.

 More features like payment integration and order tracking are coming soon!
